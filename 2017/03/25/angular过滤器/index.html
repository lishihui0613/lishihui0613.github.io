常用的过滤器

限制模型上数据变化的时间

  • 当改变数据的时候时间太快，可以限制模型上数据变化的时间

给input加个属性ng-model-options=”{updateOn:’blur’}” 当失去焦点的时候刷新；ng-model-options=”{debounce:500}” 当过500ms刷新页面

自定义过滤器

  • 通过模块来创建过滤器

  • 在控制器中使用过滤器
    过滤器的名字+Filter(属于自定义过滤器),注入进来 uppercaseFilter(数据)；比如orderBy得写成orderByFilter

  • 让首字母大写
