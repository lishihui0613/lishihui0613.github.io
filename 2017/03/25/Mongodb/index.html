Mongodb 数据库

1.什么是MongoDB

  • MongoDB是一个基于分布式文件存储的开源数据库系统
  • MongoDB 将数据存储为一个文档，数据结构由键值对(key=>value)组成。MongoDB 文档类似于 JSON 对象。字段值可以包含其他文档，数组及文档数组。
  • 是前端最佳的数据库，里面可以写js代码，开源免费，单机性能非常高每秒能读写1.6万次
  • 关系查询功能不够完善
  • 在node里用mongodb，就是mongoose

3.mongodb启动与连接 下面2个都得启动

3.1 windows启动服务器端

  • 找到mongodb安装目录,一般是 C:\Program Files\MongoDB 2.6 Standard\bin
  • 按下Shift+鼠标右键,选择在此处打开命令窗口(就是DOM命令窗口)
  • 在除C盘外的盘符新建一个空目录,如 D:\Mongodb\data

  • 在打开的命令行中输入mongod –dbpath=刚创建的空目录,如mongod –dbpath=D:\Mongodb\data 再按回车键

  • .如果出现waiting for connections on port 27017就表示启动成功,已经在27017端口上监听了客户端的请求

  • 注意：–dbpath后的值表示数据库文件的存储路径,而且后面的路径必须事先创建好，必须已经存在，否则服务开启失败，每个操作数据库的时候都得启动服务器
  • 注意：这个命令窗体绝对不能关,关闭这个窗口就相当于停止了mongodb服务

3.2 启动客户端连接服务器

  • 找到mongodb安装目录,一般是 C:\Program Files\MongoDB 2.6 Standard\bin
  • 按下Shift+鼠标右键,选择在此处打开命令窗口 ，就是和上面的路径一样，在重新打开一个
  • 命令窗体中输入 mongo –host=127.0.0.1 或者 mongo 按回车键
